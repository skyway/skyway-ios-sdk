# SkyWay iOS SDK Swift Sample

```
swift 4.0以上
Xcode 9.0以上
ios 9.0以上
Cocoapods 1.0.0以上

```

## 機能


- 1-1の映像サンプル
- 1-1のチャットサンプル

## セットアップ

### 1. ライブラリのインストール

```
$ pod install
```

### 2. APIKEY, DOMAINの書き換え

```
// AppDelegate.swift
// https://webrtc.ecl.ntt.com/からAPIKeyとDomainを取得してください
var skywayAPIKey:String? = "xxx"
var skywayDomain:String? = "xxx"

```
